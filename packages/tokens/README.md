# Design tokens (`cd-*`)

These CSS custom properties encode a **document-first** reading rhythm: calm background, warm text, tight radii, and monospace labels for “system voice”.

They are intended for static sites and prototypes. **They are not extracted from any proprietary model UI**—they are a public, reusable dark “document UI” baseline.

Consumers can copy `tokens.css` or `@import` it from a relative path in static hosting.
