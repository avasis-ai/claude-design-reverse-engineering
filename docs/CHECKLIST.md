# Authoring checklist (Claude-style IA)

Use when writing long pages, in-app help, or release notes.

| Aspect | Ask |
| --- | --- |
| Structure | Does the first screen answer the question? |
| Chunks | Can each paragraph be titled with one noun phrase? |
| Lists | Are bullets parallel in grammar and length? |
| Tables | Would a table remove repeated comparison sentences? |
| Depth | Is optional detail behind `<details>`, appendices, or links? |
| Close | Is there an explicit recap or next step? |

See `output-design-framework.md` for narrative context.
